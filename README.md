###ES6 Flux Starter App

##Dependices
* Babel
* eslint
* flux
* react
* sass
* karma
* jscs
* flow
* chai
* webpack

##Installing

```
npm install
```

##Starting the dev server

```javascript
$ npm start
```

##Running Unit Tests

```
$ npm test

```

##Running Unit Tests in watch mode

```
npm run test:watch
```

##Creating a build
```
npm run build
```
